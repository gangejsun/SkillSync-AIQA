# PPT Generator - 빠른 시작 가이드

## 5분 안에 시작하기

### 1. 설치 (1분)

```bash
pip install python-pptx
```

### 2. 첫 PPT 만들기 (2분)

Claude Code에서 다음과 같이 요청하세요:

```
"AI 개발 트렌드 PPT 10장 만들어줘"
```

Claude가 자동으로:
1. 주제 확인
2. 컬러 테마와 스타일 선택 옵션 제공
3. 10장의 콘텐츠 생성
4. PPT 파일 생성
5. 다운로드 링크 제공

### 3. 결과 확인 (1분)

생성된 PPT 파일을 열어서 확인하세요. 필요하면 추가 수정을 요청할 수 있습니다.

## 컬러 테마 미리보기

### Blue (파랑) - 기본
- 전문적, 신뢰감
- 비즈니스 프레젠테이션
- 기업 발표

### Green (초록)
- 혁신, 성장
- 기술 프레젠테이션
- 스타트업

### Purple (보라)
- 창의적, 고급스러움
- 디자인 프레젠테이션
- 크리에이티브 산업

### Orange (주황)
- 열정, 에너지
- 마케팅 프레젠테이션
- 세일즈 피칭

## 문제 해결

### Q: python-pptx 설치가 안 돼요
A: pip를 업그레이드 후 재시도
```bash
pip install --upgrade pip
pip install python-pptx
```

### Q: 한글이 깨져요
A: 맑은 고딕 폰트가 설치되어 있는지 확인

### Q: 슬라이드 수를 변경하고 싶어요
A: "슬라이드 3개 더 추가해줘" 또는 "10장을 15장으로 늘려줘"

### Q: 컬러를 바꾸고 싶어요
A: "초록색으로 다시 만들어줘"
